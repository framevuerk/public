Code:

	<!-- contents -->

API


Name
Type
Default
Description
title
String
""
Title of dialog (if it's empty, title container not rendered
content
String
""
Content of dialog (if it's empty, content container not rendered
left
null
Left position of dialog.
Note that for better result on non `null` value, you better pass this by variable and add .sync modifier.
top
null
Top position of dialog.
Note that for better result on non `null` value, you better pass this by variable and add .sync modifier.
width
"auto"
Width of dialog.
height
"auto"
Height of dialog.
autoClose
Boolean
true
Close dialog by clicking buttons
modal
Boolean
false
Close dialog by clicking overlay or by ESC key
buttons
Array
[]
Buttons of dialog.
Each item in array, directly bind to fvButton component, so it should be an Object with fvButton props and values.
first-focus-on
Boolean or Number
""
First focus target element of dialog.
If it's true, first focusable element will give focus.
If it's false, last focusable element will give focus!
Else if that's Number, fv-dialog use this as index of element to be focused.
