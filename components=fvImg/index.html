To creating image(s) element, use this component. Note that before using this, you can add HammerJs as dependency of framevuerk to allow touch actions.


Javascript:

import framevuerk from 'framevuerk'
import Hammer from 'hammerjs'
framevuerk.use('hammer', Hammer)

Code:

<fv-img></fv-img>

Description

show-buttons
Boolean
