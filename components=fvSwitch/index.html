To create switch in your application, use this! Note that before using this, you can add HammerJs as dependency of framevuerk to allow touch actions.


Javascript:

import Framevuerk from 'framevuerk'
import Hammer from 'hammerjs'
Framevuerk.use('hammer', Hammer)

Code:

<fv-switch></fv-switch>
