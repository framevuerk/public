To create full featured date-time picker based on your locale in your application, use this! Note that this is use fvInput and fvDialog.
Note that before using this, you can add IDate as dependency of framevuerk to support jalaali dates.


Javascript:

import Framevuerk from 'framevuerk'
import IDate from 'idate'
Framevuerk.use('date', IDate)

Code:

<fv-datepicker></fv-datepicker>
